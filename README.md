A simple web application that helps you find vegan recipes based on ingredients you have on hand. Built with jQuery and the Spoonacular API.

## Features

- Search for vegan recipes using multiple ingredients
- Real-time recipe suggestions
- Mobile-responsive design
- Dark mode support
- Loading states and error handling
- Recipe links to detailed instructions

## Technologies Used

- HTML5
- Modern CSS (Custom Properties, Flexbox)
- jQuery
- Spoonacular API
- Google Fonts (Roboto, Russo One)

## Usage

1. Enter at least 2 ingredients separated by commas
2. Click "Find Recipes" to search
3. Browse through suggested recipes
4. Click "View Recipe" to see full instructions

## API Limits

- Free tier: 150 requests per day
- Application limits: 10 requests per minute
- Each search returns 3 recipes
